BG3 SE Console Commands

Click on a command to copy it to your clipboard.

Items

Add item to inventory

TemplateAddTo("UUID", GetHostCharacter(), X)

Add gold

AddGold(GetHostCharacter(), X)

Set tadpole state

SetTadpoleTreeState(GetHostCharacter(), X)

1: Hesitant to use worm powers, 2: Ready to use worm powers

Add tadpoles

AddTadpole(GetHostCharacter(), X)

Add inspiration

Osi.GiveInspirationPoints(GetHostCharacter(), X, "", "")

Leveling

Add experience

Osi.AddExplorationExperience(GetHostCharacter(), X)

Set level

Ext.Entity.Get(GetHostCharacter()).AvailableLevel.Level=X

Set current experience

Ext.Entity.Get(GetHostCharacter()).Experience.CurrentLevelExperience=X

Set experience to next level

Ext.Entity.Get(GetHostCharacter()).Experience.NextLevelExperience=X
Ext.Entity.Get(GetHostCharacter()).Experience.TotalExperience=X

Respec

Osi.StartRespec(GetHostCharacter())

Other

Change appearance

Osi.StartChangeAppearance(GetHostCharacter())

Change approval

Osi.ChangeApprovalRating("COMPANION UUID", GetHostCharacter(), 0, X)

Set max party size

SetMaxPartySizeOverride(X)

Recheck party size limit

Osi.PROC_CheckPartyFull()
